# install
```zsh
git clone https://github.com/optgeo/pcdb-test.git
cd pcdb-test
yarn
```

# use
```zsh
rake -T
rake download
rake shapefile
rake geojsonseq
rake tippecanoe
rake expand
```

# vector tile specifications
- layer name: pc
- minzoom: 22
- maxzoom: 22
